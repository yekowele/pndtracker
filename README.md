# pndtracker

Example project tracker app with Vue - Vuex - Firebase. 

Edit firebase config in ``` ./src/firebase/index.js ```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
